Guia de Setup Completo

0 de 31 passos completos
1
Printify — Criar Produtos
~30 min
Criar conta no Printify
Vai a printify.com → Sign Up (grátis). Usa o email da marca.
Escolher produtos — canecas e quadros
Catalog → adiciona:
• Caneca: "Ceramic Mug 11oz" (a mais vendida; o 15oz é opcional)
• Quadros: "Framed Vertical Poster" (moldura de madeira, igual ao design do site) e/ou "Matte Vertical Poster" + "Stretched Canvas"
Criar designs base
Para cada produto, cria um design no editor do Printify. Usa o teu logo e um layout exemplo (nomes + data). Isto serve de template — o Customily vai personalizar depois.
Escolher fornecedor
Para cada produto, o Printify mostra vários fornecedores (Print Providers). Para Portugal/Europa escolhe fornecedores com produção na Europa (ex.: Print Logistic, Printify Choice EU) — shipping mais rápido e sem alfândega.
Dica: Encomenda uma amostra de cada produto para ti antes de vender. Custa pouco e garantes que a qualidade é boa.
2
Printify → WooCommerce
~10 min
Ligar Printify ao WooCommerce
No Printify: Manage Stores → Add New Store → WooCommerce. Insere o URL do teu site: ourmoment-1-1347022.ingress-erytho.ewp.live
Instalar plugin Printify no WordPress
WordPress Admin → Plugins → Adicionar Novo → pesquisa "Printify" → Instalar → Ativar. Autoriza a ligação quando pedido.
Publicar produtos
No Printify, para cada produto criado: clica "Publish" → seleciona a tua loja WooCommerce. Os produtos aparecem automaticamente no WooCommerce.
Verificar produtos no WordPress
WordPress Admin → WooCommerce → Produtos. Confirma que os produtos do Printify apareceram com imagens e preços.
3
Plugins — instalar agora
~30 min
Já tens instalado: WooCommerce, Printify e Yoast SEO. A lista abaixo é o que falta. Todos são gratuitos.
WooCommerce Stripe Gateway
Plugins → Adicionar Novo → "WooCommerce Stripe Gateway" (da Woo). Sem isto não recebes dinheiro. Cria conta em stripe.com, liga, e deixa em modo de teste até fazeres a encomenda de ensaio.
Contact Form 7
O formulário de contacto do tema é apenas visual — não envia nada. Instala o Contact Form 7, cria um formulário com os campos Nome, Email e Mensagem, e o tema passa a usá-lo automaticamente.
Complianz — banner de cookies
Obrigatório pelo RGPD antes de teres visitantes reais. Corre o wizard: faz o scan dos cookies, gera o banner e a política, e guarda os consentimentos. O tema já o estiliza com as cores da marca.
MailPoet — lista de email
Grátis até 500 subscritores. Cria um formulário com o id="1" e o bloco de newsletter aparece sozinho no footer. Configura também o email de carrinho abandonado — é a automação que mais se paga.
Google Analytics + Meta Pixel
Instala hoje, mesmo sem anunciares. Usa o Site Kit (Google) e o Meta Pixel for WooCommerce. O pixel precisa de meses a recolher dados antes de os anúncios funcionarem — instalá-lo no dia em que decides anunciar é chegar à corrida sem ter treinado.
TranslatePress — PT / EN
A versão gratuita dá uma língua adicional, que é o que precisas. Traduz o texto renderizado, por isso apanha as strings do tema sem alterar código. Decide primeiro a língua principal — mudar depois dá trabalho.
Não instales tudo de uma vez. Instala, ativa, e vê o site. Se algo partir, sabes qual foi. Plugins de cookies e de tradução são os que mais colidem com temas.
4
Customily — Personalizador
~45 min
Opcional no início. Sem o Customily ($19/mês), a personalização é feita à mão: o cliente escreve os nomes num campo de texto e envia a foto, tu montas o design no Printify antes de aprovar. Nas primeiras 30 encomendas é perfeitamente sustentável.
Criar conta no Customily
Vai a customily.com → Start Free Trial ou escolhe o plano de $19/mês. Regista com o email da marca.
Instalar plugin Customily no WordPress
WordPress Admin → Plugins → Adicionar Novo → pesquisa "Customily" → Instalar → Ativar. Ou faz download do plugin no site do Customily e faz upload manual.
Ligar Customily ao WooCommerce
No painel do Customily: Settings → Integrations → WooCommerce. Insere o URL do site + API Key do WooCommerce (WooCommerce → Settings → Advanced → REST API → Add Key).
Ligar Customily ao Printify
No Customily: Settings → Integrations → Printify. Autoriza com a tua conta Printify. Isto permite que o Customily envie os designs finalizados directamente.
Criar templates no Customily
Para cada produto: Customily → Templates → New Template. Seleciona o produto do Printify e define as zonas editáveis:
• Caneca 11oz: zona de foto ao centro + nome do casal por baixo + data. Atenção à área de impressão wrap-around (a zona útil visível é frente/verso).
• Quadro/Poster: zona de foto grande + nomes em Playfair Display + data em Montserrat, centrados em baixo (igual ao mockup do site).
• Fontes a disponibilizar: Playfair Display, Montserrat + 2-3 opções de script (ex.: Great Vibes).
Associar templates aos produtos
No Customily, liga cada template ao produto WooCommerce correspondente. Quando o cliente abre o produto na loja, o personalizador aparece automaticamente.
Testar o personalizador
Abre cada produto na loja como cliente: faz upload de uma foto, escreve nomes e data, confirma que o preview ao vivo mostra tudo na posição certa dentro da zona de impressão. Testa também no telemóvel.
Nota: O editor de templates do Customily é drag-and-drop. Demora ~15 min por produto. Eles têm tutoriais em vídeo no site.
5
Pagamentos + Shipping
~15 min
Configurar o Stripe
O plugin já foi instalado na Fase 3. Agora: WooCommerce → Definições → Pagamentos → Stripe. Cola as chaves de API, ativa Apple Pay e Google Pay, e confirma que fica em modo de teste até ao ensaio final.
Configurar shipping
WooCommerce → Settings → Shipping. O Printify calcula o shipping automaticamente, mas tens de definir zonas (Portugal, Europa, Mundial) e se ofereces portes grátis acima de X€.
Definir moeda e impostos
WooCommerce → Settings → General → moeda (EUR €). Settings → Tax → configura IVA se aplicável.
6
Páginas Legais + Extras
~40 min
Criar as páginas do menu
Páginas → Adicionar Nova. Cria três páginas com estes slugs exactos e deixa o conteúdo vazio (o tema trata do resto):
• how-it-works  • about  • contact
Apaga também a página sample-page que veio com o WordPress.
Criar páginas legais
Os textos base estão no repositório, em legal/. Cria uma página por ficheiro, com o slug indicado:
• terms — Terms & Conditions
• privacy — Privacy Policy
• returns — Returns & Refunds
Substitui todos os campos entre [colchetes] pelos teus dados. Assim que existirem, aparecem sozinhas no footer.
Correr o wizard do Complianz
Instalado na Fase 3. Corre o wizard só depois de teres o Stripe e o Analytics ativos, para que o scan encontre os cookies reais. Gera o banner e a política de cookies, e guarda os consentimentos.
Configurar email
WooCommerce → Settings → Emails. Personaliza os emails de confirmação de encomenda, shipping, etc. Muda o "From" para contact@ourmoment.com.
Configurar o Yoast SEO
Já instalado. Corre o wizard, define o nome e o logo da marca, e escreve o título e a meta-descrição de cada produto. Liga o Google Search Console para veres que pesquisas te encontram.
Atenção: os textos legais em legal/ são um ponto de partida, não aconselhamento jurídico. A excepção ao direito de livre resolução em produtos personalizados é a parte mais sensível deste modelo — vale a pena confirmar com um advogado ou contabilista antes de lançar.
7
Lançamento
~10 min
Testar encomenda completa
Faz uma encomenda de teste: personaliza um produto → adiciona ao carrinho → checkout com Stripe test mode → confirma que o Printify recebe a encomenda com o design correcto.
Domínio + SSL + Go Live
Compra domínio (ourmoment.com ou .pt). No painel do hosting, aponta o domínio para o site. Ativa SSL (HTTPS). Tira o Stripe do test mode. Publica nas redes sociais.
Pronto! Quando este último passo estiver feito, a loja está live e a receber encomendas. Volta aqui quando quiseres — posso ajudar com qualquer passo.
